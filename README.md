# 🛒 Retail Sales & Profit Optimization

## 📌 Objective

Analyze retail data to identify loss-making products and improve profitability.

## 🧠 Problem Statement

A retail company is facing losses. Analyze transactions to detect issues and improve performance.

## 📊 Dataset

* Online Retail Dataset (UCI)

## 🎯 Key Questions

* Which products cause losses?
* Are there anomalies?
